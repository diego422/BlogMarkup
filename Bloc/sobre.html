Sobre mí

¡Hola! Soy Diego López, apasionado por el desarrollo web, la tecnología y la creación de experiencias digitales accesibles y fáciles de usar.

Este blog nace con la idea de demostrar que es posible crear proyectos atractivos, rápidos y funcionales usando únicamente HTML y CSS, sin depender de JavaScript para todo. Aquí comparto artículos, ideas y ejemplos que te ayudarán a mejorar tus habilidades de maquetación y diseño web.

“El código limpio no es solo para las máquinas, también es un regalo para quienes lo leen después.”

¿Qué encontrarás en BlogMarkup?

  • Artículos prácticos sobre HTML y CSS.
  • Buenas prácticas de diseño y accesibilidad.
  • Plantillas y recursos listos para usar.

Mi objetivo

Quiero inspirarte a crear proyectos web simples pero potentes, con especial atención en la experiencia del usuario y la optimización para todos los dispositivos.
